## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Lokesh-reddy18/Chess-frontend.git
cd frontend
```

2. Install Dependencies
```bash
npm install
```

3. Start Development Server
```bash
npm run dev
```

4. Build for Production
```bash
npm run build
```
